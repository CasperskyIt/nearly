**dog-create.component.ts:**
- Standalone component
- Imports: `ReactiveFormsModule`, `MatFormFieldModule`, `MatInputModule`, `MatButtonModule`, `MatIconModule`, `MatDatepickerModule`, `MatNativeDateModule`, `MatSnackBarModule`, `RouterLink`
- Inject: `DogService`, `Router`, `MatSnackBar`, `ThemeService`
- Create a `FormGroup` with:
  - `name: FormControl<string>` — required, Validators.required
  - `breed: FormControl<string>` — optional
  - `dateOfBirth: FormControl<Date | null>` — optional (Material DatePicker binds to Date)
- `submitting = signal(false)` — disable button during submission
- Method `async onSubmit()`:
  1. If form invalid, mark all as touched, return
  2. Set `submitting(true)`
  3. Build `CreateDogRequest`: `{ name: form.value.name!, breed: form.value.breed || undefined, date_of_birth: form.value.dateOfBirth ? form.value.dateOfBirth.toISOString().split('T')[0] : undefined }`
  4. Call `dogService.createDog(request)`
  5. If result is not null: show success snackbar "Dog added" (per D-32), navigate to `/${appPrefix}/dogs/${result.id}` (go to detail page)
  6. If result is null: show error snackbar "Failed to add dog" (per D-31)
  7. Set `submitting(false)` in finally
- Method `onCancel()`: navigate to `/${appPrefix}/dogs`
