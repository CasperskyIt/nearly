```typescript
import { Injectable, inject, signal } from '@angular/core';
import { SupabaseService } from './supabase.service';
import { LoggerService } from './logger.service';
import { Dog, CreateDogRequest, UpdateDogRequest, DogGuardian } from '../models/dog.model';

@Injectable({ providedIn: 'root' })
export class DogService {
  private supabaseService = inject(SupabaseService);
  private logger = inject(LoggerService);

  // State signals (per D-28)
  currentDog = signal<Dog | null>(null);
  dogs = signal<Dog[]>([]);
  loading = signal(false);
  error = signal<string | null>(null);
```

**Methods to implement:**

1. `async getDogs(): Promise<Dog[]>` — Query `dogs` table (RLS automatically filters to user's accessible dogs). Set `loading` signal during fetch. On success, update `dogs` signal. On error, set `error` signal, log via LoggerService. Return the dogs array.

2. `async createDog(request: CreateDogRequest): Promise<Dog | null>` — Insert into `dogs` table with `owner_id: (await this.supabaseService.supabase.auth.getUser()).data.user?.id`. Include only non-undefined fields. On success: add to `dogs` signal array, set as `currentDog`, return dog. On error: set `error` signal, return null.

3. `async updateDog(id: string, request: UpdateDogRequest): Promise<Dog | null>` — Update `dogs` table where `id` matches. Add `updated_at: new Date().toISOString()`. On success: update dog in `dogs` signal array (map and replace), update `currentDog` if it was the updated dog. Return updated dog. On error: set `error` signal, return null.

4. `async deleteDog(id: string): Promise<boolean>` — Delete from `dogs` where `id` matches (CASCADE handles dog_guardians, care_events, etc.). On success: remove from `dogs` signal array (filter), clear `currentDog` if it was the deleted dog, return true. On error: set `error` signal, return false.

5. `async uploadAvatar(dogId: string, file: File): Promise<string | null>` — Per D-29: Upload to `dog-avatars/{dogId}` using `this.supabaseService.supabase.storage.from('dog-avatars').upload(dogId, file, { upsert: true, contentType: file.type })`. On success: get public URL via `.getPublicUrl(dogId)`, then call `updateDog(dogId, { avatar_url: publicUrl })`, return the URL. On error: set `error` signal, return null. Add cache-busting query param `?t=${Date.now()}` to the public URL to avoid stale browser cache.

6. `setCurrentDog(dog: Dog | null): void` — Update `currentDog` signal. If dog is not null, store `dog.id` in `sessionStorage.setItem('currentDogId', dog.id)` for session persistence (per D-03).

7. `async restoreCurrentDog(): Promise<void>` — Read `sessionStorage.getItem('currentDogId')`. If exists and `dogs()` is populated, find matching dog and call `setCurrentDog()`. If `dogs()` is empty, call `getDogs()` first, then find and set.

8. `async getGuardians(dogId: string): Promise<DogGuardian[]>` — Query `dog_guardians` where `dog_id = dogId` and `status = 'accepted'`. Return array. On error: set `error` signal, return empty array.

**Error handling pattern (mirrors SupabaseService):**
```typescript
try {
  this.loading.set(true);
  this.error.set(null);
  // ... operation
} catch (err: any) {
  this.error.set(err.message);
  this.logger.error('DogService error:', err);
  // return null/false/[]
} finally {
  this.loading.set(false);
}
```

All Supabase queries use `this.supabaseService.supabase.from('dogs')` pattern (accessing the client directly, same as SupabaseService does internally).
