**Step 1: Create `frontend/src/app/services/map.service.ts`**

An injectable service that owns Leaflet map state. Per RESEARCH.md Pattern 2, the service receives a DOM container ID from the component.

The service must contain:
- `private map: L.Map | null = null`
- `private markers: L.Marker[] = []`
- `private tileLayer: L.TileLayer | null = null`
- `private lightTiles = 'https://{s}.basemaps.cartocdn.com/rastertiles/voyager/{z}/{x}/{y}{r}.png'`
- `private darkTiles = 'https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png'`
- `initMap(containerId: string, options?: { center: [number, number], zoom: number }): void` — creates the L.Map with `zoomControl: false`, default center `[52.2297, 21.0122]`, zoom 14. Stores in `this.map`.
- `destroyMap(): void` — calls `this.map.remove()` if map exists, nulls references
- `getMap(): L.Map | null` — returns the map reference (needed for `setView`, `getBounds`, `invalidateSize`, `zoomIn`, `zoomOut`, event listeners)
- `updateTileLayer(isDark: boolean): void` — sets tile URL to dark or light tiles
- `setTileLayer(isDark: boolean): void` — creates and adds the initial tile layer with attribution `'&copy; OpenStreetMap contributors &copy; CARTO'` and `maxZoom: 19`
- `clearMarkers(): void` — removes all markers from map, empties array
- `addMarker(lat: number, lng: number, icon: L.DivIcon, onClick: () => void): void` — creates marker, adds to map, stores in array
- `invalidateSize(): void` — calls `this.map?.invalidateSize()`

Import Leaflet as `import * as L from 'leaflet';`. Use `@Injectable({ providedIn: 'root' })`.

**Step 2: Slim HomeComponent to orchestrator (D-07)**

Rewrite `frontend/src/app/pages/home/home.component.ts` to be under 100 lines. It becomes a thin wiring layer:

```typescript
import { Component, signal, computed, inject } from '@angular/core';
import { ThemeService } from '../../config/theme.service';
import { I18nService } from '../../config/i18n.service';
import { Language, languageNames } from '../../config/i18n.config';
import { MapComponent } from '../../components/map/map.component';
import { PlacesListComponent, Place } from '../../components/places-list/places-list.component';

@Component({
  selector: 'app-home',
  standalone: true,
  imports: [MapComponent, PlacesListComponent],
  templateUrl: './home.component.html',
  styleUrl: './home.component.scss'
})
export class HomeComponent {
  private themeService = inject(ThemeService);
  private i18nService = inject(I18nService);

  // State signals
  selectedPlace = signal<Place | null>(null);
  selectedCategories = signal<Set<string>>(new Set());
  showLangMenu = signal(false);
  showList = signal(true);
  isLoading = signal(false);
  places = signal<Place[]>([]);
  visiblePlaces = signal<Place[]>([]);

  // Computed
  darkMode = computed(() => this.themeService.darkMode()());
  themeStrings = computed(() => this.themeService.themeStrings());
  themeIcons = computed(() => this.themeService.themeIcons());

  get t() { return this.i18nService.t; }
  get currentLanguage(): Language { return this.i18nService.getCurrentLanguage(); }
  get languages() {
    return Object.entries(languageNames).map(([code, name]) => ({
      code: code as Language, name
    }));
  }
  get categories() {
    const icons = this.themeIcons();
    return Object.keys(icons.categories).map(key => ({
      id: key,
      label: this.t.categories[key] || key,
      icon: icons.categories[key]
    }));
  }

  // Event handlers — delegate to child components or update local state
  toggleCategory(id: string): void {
    this.selectedCategories.update(set => {
      const newSet = new Set(set);
      newSet.has(id) ? newSet.delete(id) : newSet.add(id);
      return newSet;
    });
  }

  onPlacesLoaded(places: Place[]): void {
    this.places.set(places);
  }

  onPlaceSelected(place: Place): void {
    this.selectedPlace.set(place);
  }

  onVisiblePlacesChanged(places: Place[]): void {
    this.visiblePlaces.set(places);
  }

  closePlaceDetail(): void {
    this.selectedPlace.set(null);
  }

  toggleList(): void {
    this.showList.update(v => !v);
  }

  toggleLangMenu(): void {
    this.showLangMenu.update(v => !v);
  }

  setLanguage(lang: Language): void {
    this.i18nService.setLanguage(lang);
    this.showLangMenu.set(false);
  }

  toggleMenu(): void {
    // No-op for now — menu functionality deferred
  }
}
```

This is approximately 70-80 lines — well under 100.
